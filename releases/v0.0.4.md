# v0.0.4 — Telemetry endpoint fix

**June 2026**

A small fix release.

## Fixes

- **Anonymous instance ping restored.** The telemetry endpoint moved to the stable custom domain `velvet.aroundmyroom.com`. The previous `velvet-velvet.aroundmyroom.workers.dev` subdomain had been renamed and was returning 404, so boot pings were silently failing. Using the branded custom domain keeps it working even if the worker subdomain changes again.

## Docs

- Added a one-time **"Switching from the old mStream repo"** guide to [docs/install.md](https://github.com/aroundmyroom/Velvet/blob/main/docs/install.md) for bare-metal / Node.js installs. Because the Velvet repo has a fresh git history, a plain `git pull` fails — the guide shows how to repoint an existing clone in place (config and library are preserved automatically) or clone fresh and carry state over.

## Docker

```shell
docker pull ghcr.io/aroundmyroom/velvet:v0.0.4
```
