# v0.0.1 — Initial Release

**June 2026**

**Velvet** — a music server, evolved with AI slop. Enter the danger zone.

## Highlights

- Self-hosted music streaming with multi-user support
- Subsonic API compatibility (works with Feishin, Symfonium, etc.)
- Docker image: `ghcr.io/aroundmyroom/velvet:v0.0.1` (multi-arch: amd64 + arm64)
- Album browser, smart playlists, ReplayGain, BPM/harmonic mixing
- Auto-DJ with Last.fm similar-artist seeding
- ListenBrainz and Last.fm scrobbling
- Tag editor, YouTube download, Internet radio recording
- DLNA/UPnP and Sonos support
- 8-band EQ, crossfade, gapless playback
- i18n: 12 languages

## Docker

```shell
docker pull ghcr.io/aroundmyroom/velvet:v0.0.1
```

See [docs/docker.md](https://github.com/aroundmyroom/Velvet/blob/main/docs/docker.md) for full setup instructions.
