# Velvet v0.0.9

Maintenance release.

## Build & maintenance
- **Version sync now covers every webapp cache-buster.** The release version stamper (`scripts/sync-webapp-version.cjs`) now updates any `.js`/`.css` `?v=` cache-buster from `package.json` across every webapp entry document. The mobile stylesheet cache-buster (`mobile/app.css?v=`) was previously left un-bumped, so mobile users could keep loading a stale cached stylesheet after upgrading. All cache-busters are once again single-sourced from `package.json`.
