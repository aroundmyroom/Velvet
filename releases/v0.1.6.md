# Velvet v0.1.6 — Release tooling hardening

A small maintenance release that makes the one-command release flow robust.

## Maintenance
- **`npm run release` rebases before pushing.** The release script now runs
  `git pull --rebase origin main` before `git push`, so a release no longer fails
  when `main` has advanced (e.g. a Dependabot merge) since the local commit was
  made — concurrent commits are integrated automatically instead of the push being
  rejected.
