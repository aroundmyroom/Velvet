# v0.0.4 — Telemetry, scanner hardening, and cleanup

**June 2026**

A maintenance release with one production fix, one Docker-scan hardening pass, and repository cleanup.

## Fixes

- **Anonymous instance ping restored.** The telemetry endpoint moved to the stable custom domain `velvet.aroundmyroom.com`. The previous `velvet-velvet.aroundmyroom.workers.dev` subdomain had been renamed and was returning 404, so boot pings were silently failing. Using the branded custom domain keeps it working even if the worker subdomain changes again.
- **Scanner timeout guard extended to all metadata backfill phases.** Docker scans that hit a single bad FLAC/metadata file during targeted re-parse no longer hang forever. The scanner now applies the same 30-second timeout to art, cue, duration, tech-meta, and album-version refreshes, then records the failure and continues.
- **Timeout logging made actionable.** When a file does time out, the scanner log now includes the exact phase, vpath, scan ID, and file path so the blocking file can be found quickly on the affected system.

## UI

- **Player branding scaled up.** The sidebar logo and centre VU logo were increased in size so the Velvet branding is clearer in the main player.

## Docs

- Added a one-time **"Switching from the old mStream repo"** guide to [docs/install.md](https://github.com/aroundmyroom/Velvet/blob/main/docs/install.md) for bare-metal / Node.js installs. Because the Velvet repo has a fresh git history, a plain `git pull` fails — the guide shows how to repoint an existing clone in place (config and library are preserved automatically) or clone fresh and carry state over.
- Clarified in [docs/docker.md](https://github.com/aroundmyroom/Velvet/blob/main/docs/docker.md) that `ghcr.io/aroundmyroom/velvet:latest` can be used directly as well as version-pinned tags.

## Cleanup

- Removed the unused legacy `webapp/alpha` frontend files from the repository.

## Docker

```shell
docker pull ghcr.io/aroundmyroom/velvet:v0.0.4
```

See [docs/docker.md](https://github.com/aroundmyroom/Velvet/blob/main/docs/docker.md) for full setup instructions.
